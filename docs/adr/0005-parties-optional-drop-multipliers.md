# Parties are optional drop multipliers

The product spine is solo-capable: watch a Fixture, mint Moments, Micro-Play, open Packs, Craft, Duel. Multiplayer "Rooms" are reframed as optional Parties that multiply drops and host friend Duels. Existing SSE/runtime infra is reused; Party is not required to complete the core loop. Product name stays Final Whistle Rooms for submission continuity.
